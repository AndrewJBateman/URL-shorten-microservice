URL Shortener Microservice


Enter a URL after 'new/' to create a short URL - see example below

Example: https://url-shorten-microservice-ajb.glitch.me/new/www.google.com

This will return:

{ "_id": "5b0917bdee1ac706d2f4cd05",
"originalURL": "www.google.com",
"shortenedURL": "HJSaPc8ym"
}

Using the shortened url, redirects back to

https://www.google.com
